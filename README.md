# chatPager

chatPager is an experimental browser extension for Chrome and Edge that adds pagination and content-packing controls to long ChatGPT conversations on `chatgpt.com`.

It injects a floating toolbar into the page, lets you switch between multiple hide modes (`display`, `detach`, `compress`, `referred compress`), and is mainly intended for debugging, DOM experiments, and performance testing on long conversation threads.

## What this is

- A **local browser extension**
- Targeting **ChatGPT web (`chatgpt.com`)**
- Built for **manual loading in Chrome/Edge**
- Focused on **pagination, DOM packing, and UI/debug experiments**

## Main features

- Paginate long conversations
- Hide non-current pages
- Pack hidden message content with different strategies
- Apply packing to selected page ranges
- Debug-oriented floating toolbar and settings panel
